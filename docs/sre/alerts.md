# Alerts → Runbook Mapping

Map production alerts to runbooks here. Each alert should include a direct link to the runbook and an on-call severity mapping.

Example mapping
- `API_5xx_Spike` → `docs/sre/runbooks/api-5xx-spike.md` (SEV2/SEV1 escalation rules included)

Add entries for new alerts and ensure the alert description includes this mapping.
